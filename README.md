# Medical Pharma Project

## Description
This is a web application for managing a medical shop.  
It allows users to register, login, manage medicines, track stock, and view dashboard statistics.  
This project is built with **Python**, **Flask**, and **MySQL**.

---

## Features
- ✅ User Registration & Login with secure password hashing  
- ✅ Dashboard showing stock and sales overview  
- ✅ Add, Update, Delete medicines  
- ✅ Search and filter medicines by name or expiry date  
- ✅ Track medicine quantity and expiry  
- ✅ Responsive UI using HTML & Bootstrap

---

## Tech Stack
- **Backend:** Python, Flask  
- **Database:** MySQL  
- **Frontend:** HTML, CSS, Bootstrap  
- **Security:** Werkzeug password hashing
